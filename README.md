StyleflashereZPlatformBaseBundle
================================

Base bundle for ezpublish 5 and ezplatform projects.

# Features

* Custom content fetch services: filter, sort and fetch children only by defining some services in yml.

* Sujetcontroller

* Search route

* E-mail obfuscation

* Youtube twig filter
